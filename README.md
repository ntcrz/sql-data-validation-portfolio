# SQL Data Validation Portfolio

## Overview

This project demonstrates SQL data validation techniques used during Quality Assurance testing. The examples are based on the Chinook sample database and represent common backend validation scenarios performed by QA analysts.

## Objectives

- Validate backend data using SQL
- Verify business rules
- Detect data integrity issues
- Practice SQL queries used during functional and regression testing

## Tools Used

- Microsoft SQL Server
- SQL
- Chinook Sample Database
- GitHub
